ИНСТРУКЦИИ ПО НАСТРОЙКЕ ПРОЕКТА
================================

1. НАСТРОЙКА БАЗЫ ДАННЫХ:
   - Установите SQL Server
   - Создайте базу данных EducationalPlatform
   - Выполните скрипты в следующем порядке:
     * CompleteDatabaseSchema.sql
     * AdditionalTables.sql
     * FixDatabaseIssues.sql
     * InitializeTestData.sql

2. НАСТРОЙКА ПРОЕКТА:
   - Откройте проект в Visual Studio
   - Восстановите пакеты NuGet
   - Обновите строку подключения в appsettings.json

3. ЗАПУСК ПРИЛОЖЕНИЯ:
   - Выберите целевую платформу (Windows, Android, iOS)
   - Нажмите F5 для запуска

4. ТЕСТОВЫЕ ДАННЫЕ:
   - Админ: admin / admin123
   - Учитель: teacher / teacher123
   - Студент: student / student123

5. ОСНОВНЫЕ ФУНКЦИИ:
   - Регистрация и авторизация
   - Создание и прохождение курсов
   - Система достижений и валюты
   - Магазин для кастомизации профиля
   - Поддержка и обратная связь
   - Многоязычность (русский/английский)

6. РОЛИ ПОЛЬЗОВАТЕЛЕЙ:
   - Student: прохождение курсов, покупки в магазине
   - Teacher: создание курсов, управление группами
   - Admin: полный доступ к системе
   - ContentManager: управление контентом и новостями
